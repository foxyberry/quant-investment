## 골든크로스/데드크로스 기능

### 파일
- `screener/conditions/ma.py` - MACrossUpCondition, MACrossDownCondition
- `scripts/screening/korean_crossover.py` - 실행 스크립트

### 사용법
```bash
# 기본 실행 (20일선 vs 60일선, 최근 5일)
python scripts/screening/korean_crossover.py

# 이평선 기간 변경
python scripts/screening/korean_crossover.py --short-ma 5 --long-ma 20

# 감지 기간 변경 (최근 10일 내 발생)
python scripts/screening/korean_crossover.py --lookback 10
```

### 출력 정보
- 골든크로스 발생 종목 (상승 추세 전환)
- 데드크로스 발생 종목 (하락 추세 전환)
- 상승/하락 추세 강도 순위
