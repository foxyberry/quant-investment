# quant-investment
quant-investment

## stack
- Backtrader (백테스팅 + 실전)
- yfinance (데이터)
- ib_insync (IBKR 연동)
- pandas/numpy (데이터 처리)


## how to install
```
pip install 'backtrader[plotting]' matplotlib pandas numpy

pip install --upgrade yfinance --no-cache-dir


## 실전용
pip install ib_insync
```

버전 확인
```
pip show yfinance                            
Name: yfinance
Version: 0.2.63
```
