## 배경

- `lib/` 폴더가 `screener/`, `engine/`과 완전히 중복됨 (사용되지 않음)
- `scripts/legacy/`에 레거시 코드가 방치되어 있음
- `visualizer/`가 레거시 코드에서만 사용됨
- `screener/base.py` 추상 클래스가 상속되지 않음
